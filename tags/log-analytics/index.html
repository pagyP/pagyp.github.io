Azure Security Center &amp; log Analytics Workspaces

Azure Security Center is a good thing to have as part of your Azure resources and it comes in two tiers: Free or Standard. By default it is enabled in your Azure subscription at the free tier and changing that to standard unlocks additional features and comes with some costs . So you’ve upgraded Security Center to standard and you have enabled data collection and you chose the option ‘Use workspace(s) created by Security Center (default)’.
